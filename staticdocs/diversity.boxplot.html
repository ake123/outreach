diversity.boxplot

Usage

diversity.boxplot(dat, sample.groups, diversity.index = "shannon", det.th = NULL,
  title = NULL, col.list = NULL)

Arguments

dat
data matrix in the original (non-log) scale: features x samples.
sample.groups
specify the distinct sample groups
diversity.index
Options: "shannon" / "invsimpson" / "richness" / "evenness"
det.th
Detection threshold for richness and evenness calculations
title
figure title
col.list
Optional: colors for the sample groups Returns:

Value

Sample group list corresponding to the boxplot groups.
