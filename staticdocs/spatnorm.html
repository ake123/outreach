2D Spatial normalisation of microarrays

Usage

spatnorm(data, x = "x", y = "y", signal = "signal", method = c("loess"), span = 0.03,
  degree = 2, family = "symmetric", subset = 1 == 1)

Arguments

data
dataframe with positional and signal information
x
name of the x column in the dataframe
y
name of the y column in the dataframe
signal
name of the signal column in the dataframe
method
method name, currently only "loess" is supported
span
span
degree
degree
family
family
subset
logical expression indicating rows to use for normalisation
