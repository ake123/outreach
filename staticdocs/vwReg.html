Description: Draw regression curve with smoothed error bars based on the Visually-Weighted Regression by Solomon M. Hsiang; see http://www.fight-entropy.com/2012/07/visually-weighted-regression.html The R implementation is based on Felix Schönbrodt's code (under MIT/FreeBSD license) from http://www.nicebread.de/visually-weighted-watercolor-plots-new-variants-please-vote/
